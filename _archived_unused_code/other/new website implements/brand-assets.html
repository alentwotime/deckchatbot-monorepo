Implementation Guidelines

Brand Consistency Rules

  • Always use the official color palette
  • Maintain proper contrast ratios for accessibility
  • Use Orbitron font for headings, Roboto for body text
  • Keep the robot character recognizable at all sizes
  • Apply neon glow effects consistently

Technical Specifications

  • Logo minimum size: 120px width for horizontal layout
  • Clear space: Minimum 1x logo height on all sides
  • File formats: SVG for web, PNG for raster, PDF for print
  • Color profiles: sRGB for digital, CMYK for print
  • Animation duration: 0.3s for interactions, 2s for ambient effects

Usage Restrictions

  • Do not alter the robot character's proportions
  • Do not use the logo on busy backgrounds without proper contrast
  • Do not separate the robot head from the text without approval
  • Do not use colors outside the approved palette
  • Do not apply effects that compromise legibility
